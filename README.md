# apis-hub
A web app made with Vue.js that lets people try out a selection of APIs.  

__Current working APIs:__
- Pictures
  - Random.dog
  - Lorem Picsum
- TV / Movies
  - OMDb
  - TMDb
- Dictionaries
  - Owlbot
  - Merriam-Webster
